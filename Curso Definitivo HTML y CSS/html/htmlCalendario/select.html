Select

Para poder hacer un Select existen dos opciones:

  1. Forma con pocas opciones mediante la etiqueta select
  2. Forma con muchas opciones mediante la etiqueta input list
Primera forma (Pocas opciones)

Esta es usada en el momento en que las opciones de selección no son muchas, de forma que el usuario pueda hacer scroll de manera moderada.

Segunda forma (Muchas opciones)

Esta es usada al momento en que las opciones de selección son demasiadas, de forma que el usuario pueda escribir y a medida que este digita, las opciones vayan apareciendo. Esto es bueno ya que evita el cansancio de estar haciendo scroll por mucho tiempo
